# Assets del tema Ecos de Rumiñahui

Copia aquí la carpeta `assets` de la landing page original (imágenes, hojas de estilo personalizadas y librerías JavaScript). La estructura ya está creada para que puedas colocar cada archivo en su directorio correspondiente:

- `css/`
- `js/`
- `img/`
- `docs/`
- `vendor/`

Cada subcarpeta contiene un archivo `.gitkeep` únicamente para mantener la estructura en el control de versiones; puedes eliminarlo cuando agregues los recursos definitivos.
